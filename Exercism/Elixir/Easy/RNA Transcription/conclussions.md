# Conclussions
1. To get integer representation of a char you need to put question mark before char, i.e. `?A == 65` and `?a == 97`
2. charlist is defined with `~c""` symbol, i.e. `~c"awsdfs"` is equal to `[?a, ?w, ?s, ?d, ?f, ?s]`
